# Nag

Nag is a chore and reminder app with a Rust backend and React frontend.

It helps you track recurring chores, see what is due, and receive Telegram notifications (and possibly other notification channels in the future) with an inline **Mark done** action.

## Features

- Create, update, complete, and delete chores
- Cron and interval-based schedules
- Due/overdue calculation with upcoming support
- Tags for organizing chores
- Completion history
- OIDC auth support (optional)
- Telegram notifications for due chores
- Inline Telegram button to mark chores complete
